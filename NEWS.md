# fcmfd 0.1.2

* Added a global view mode (`view = "global"`) to `plot_prototypes()` to visualize all cluster prototypes combined in a single plot.
* Renamed `cluster_quality()` to `membership_quality()` to better reflect its focus on auditing fuzzy membership matrices.
* Added a logical `show_title` parameter to all plotting functions to programmatically control title visibility.
* Added the `students_wellbeing` real-world dataset to provide authentic examples for fuzzy clustering applications.
* Explicitly documented all features in the `sim_likert7` dataset to resolve code-documentation consistency (codoc) warnings.
